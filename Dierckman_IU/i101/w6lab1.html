Frequent Terms

  • Process: How we do/acomplish something.

  • Program: Specific language that allows a process to occur.

  • Algorithm: Set of rules/discription of the process.

  • To Call: Basically to have the program pull up something that is stored in its memory.

  • Command: Does something

  • Function: A full line of code that makes something happen.

  • Strings: Any combination of letters, numbers, ect.. that are enclosed by "quotations".

  • Variable: Symbolic name used to store and represent data/information to be used somewhere within a program.

  • Naming: Creating names is crucial, should be about what is being created.

JES Python Terms

  • Program Area: Where you can write code in functions. Code written here can be saved out to disc and loaded into memory where it can be accessed in the interactions area.

  • Interactions Area: You can write code here aswell, but it will not be saved to disc. You can call functions here that are defined in the program area.

  • File Menu: Allows you to manage code files.

  • Edit Menu: Basic edditing comands, just like you would see in any application.

  • Watcher Menu: Allows you to watch the value of variables during program execution. Helps with debbugging.

  • Media Tools Menu: Contains tools that help us examine aspects of the different media.

  • JES Functions Menu: Contains directory of various functions supported inside JES. Selecting a function from this menu will add it to your code in the program area and display info about the function in the help area.

  • Window Layout Menu: This menu allows you to change the windows shown in your application
